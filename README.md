# Task-Management-System

A simple console-based **ToDo List Application** developed using Java.  
This project allows users to manage tasks efficiently with features like adding, viewing, and categorizing tasks.

---

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ⭐ View tasks based on priority
- 📂 View tasks category-wise
- ❌ Exit option
- 🖥️ Menu-driven console interface

---

## 🛠️ Technologies Used

- Java
- OOP Concepts (Classes & Objects)
- ArrayList / Collections
- Scanner (User Input)
